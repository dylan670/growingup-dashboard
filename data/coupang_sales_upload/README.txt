쿠팡 판매 CSV 업로드 폴더 (로켓배송 제품별 판매)
==================================================

[왜 이 폴더가 필요한가]
쿠팡 로켓배송(벤더 풀필먼트)은 쿠팡이 상품을 매입해서 직접 판매하는
구조이므로, 판매자 Wing Open API 로는 개별 주문 조회가 불가능합니다.
대신 Supplier Hub 매출/정산 리포트 CSV 를 수동 다운로드 → 여기 드롭 →
자동 파싱해서 orders.csv 에 누적 병합.

판매자 판매(Wholesale) / 로켓그로스 주문은 쿠팡 Wing API 로 이미 자동
수집되므로, 이 폴더에는 주로 "로켓배송 상품별 판매" 리포트만 업로드.

[다운로드 경로 — 쿠팡 Supplier Hub]
주의: wing.coupang.com (판매자 Wing) 이 아니라
      supplier.coupang.com (벤더 Supplier Hub) 입니다.

1. https://supplier.coupang.com/ 로그인 (벤더 계정)
2. 상단 메뉴: 애널리틱스 (Analytics)
3. 하위 메뉴: 판매 분석 / 매출 분석 / 상품별 실적 중 택1
   (메뉴 이름 미세 변경 가능 — "일자별 상품 판매" 류 리포트)
4. 기간: 최근 30일 권장 / 기간 단위: "일별" 필수
5. 엑셀/CSV 다운로드 → 이 폴더에 드롭

[대안 경로]
Supplier Hub 의 메뉴 구성이 변경되어 판매 리포트를 찾기 어려우면:
  - 정산 → 정산내역 상세 (일별/상품별 내역)
  - 상품 → 상품별 통계 / 매출
  - 홈 → 대시보드 내 "판매 리포트 다운로드"

어떤 메뉴든 일별 × 상품별로 쪼개진 리포트면 자동 파싱 가능.

[필수 컬럼 (이름 유연 매칭)]
반드시 필요:
- 날짜 / 일자 / 집계일 (YYYYMMDD 또는 YYYY-MM-DD)
- 상품명 / 제품명 / 옵션명
- 매출 / 판매금액 / 정산금액 / 실결제금액

선택:
- 판매수량 / 수량 (없으면 1 로 계산)
- 주문번호 (없으면 자동 해시 생성)
- 배송방식 (로켓배송/로켓그로스/업체배송 — 참고용)

[자동 브랜드 분류]
상품명 키워드 기반 자동 분류:
- "김똑똑", "떡뻥", "똑똑연구소" → store='쿠팡_똑똑연구소'
- "롤라루", "캐리어", "백팩", "여행" → store='쿠팡_롤라루'
- "오즈키즈", "공룡의꿈", "침낭" 등 → store='쿠팡' (미분류)
  → 필요 시 utils/products.py 의 BRAND_RULES 에 키워드 추가

[누적 병합 동작 — 매주 30일 CSV 업로드 패턴]
새 파일의 날짜 범위에 해당하는 기존 쿠팡 데이터만 교체,
그 외 날짜의 기존 데이터는 보존.

예시:
  주차 1 (04-23): 03-24 ~ 04-22 (30일) → orders.csv 에 30일 저장
  주차 2 (04-30): 03-31 ~ 04-29 (30일)
    → 03-31~04-22 교체 (수치 보정)
    → 03-24~03-30 보존 (기존 주문 유지)
    → 04-23~04-29 신규 추가
  점진적으로 90일+ 히스토리 축적

[실행]
수동: .venv\Scripts\python.exe scripts\sync_coupang_sales_csv.py
자동: sync_all.bat 에 포함 (매일 오전 10시)

[주의]
- 개인정보 없음 (상품명 + 수량 + 금액만)
- 원본 CSV 는 .gitignore 로 public repo 에서 제외
- 파싱 결과(orders.csv)만 Streamlit Cloud 반영
- 발주리스트(PO) 는 "쿠팡이 벤더에게 주문"한 것이므로 매출 아님 — 혼동 주의
  → 반드시 "판매 분석" 또는 "매출 분석" 메뉴에서 다운로드

[Supplier Hub 네비게이션 참고 — 2026년 기준]
  홈 | 상품 | 물류 | 정산 | 애널리틱스 ← 여기!  | 마이샵 | 광고 관리 | ...

  물류 → 발주리스트 (×) : 이건 쿠팡의 발주 요청, 매출 아님
  정산 → 정산내역 (△) : 가능하지만 통합 금액
  애널리틱스 → 판매 분석 (○) : 일별 × 상품별 판매 — 권장
