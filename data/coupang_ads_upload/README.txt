쿠팡 광고 CSV/Excel 업로드 폴더
================================

[자동 병합 순서]
1. 쿠팡 광고센터 접속: https://advertising.coupang.com/
2. 리포트 → 광고 리포트 → 기간 선택 (최근 7일 권장)
3. 다운로드: CSV 또는 Excel 형식
4. 받은 파일을 이 폴더에 그냥 드롭 (파일명 상관없음)
5. 실행: .venv\Scripts\python.exe scripts\sync_coupang_ads_csv.py

[매일 자동 실행]
sync_all.bat 에 자동 추가됨 → 매일 오전 10시 이 폴더 확인.
파일 없으면 스킵, 있으면 자동 파싱.
(쿠팡 광고 Open API 미지원 → 매주 1회 정도 수동 다운로드)

[캠페인 → 브랜드 자동 분류]
- "김똑똑" "떡뻥" "로켓그로스" → 쿠팡_똑똑연구소
- "AI 광고" "롤라루" "캐리어" "백팩" → 쿠팡_롤라루
- 기타 → 쿠팡 (미분류 — utils/products.py 에 키워드 추가)

[지원 포맷]
- .xlsx / .xls (엑셀)
- .csv (UTF-8-sig, cp949, euc-kr 자동 탐지)

[필수 컬럼 (이름 유연 매칭)]
- 날짜 / 기간 / 일자
- 캠페인명 / 캠페인 / 광고상품명
- 광고비 / 비용
- (선택) 노출수, 클릭수, 주문수, 매출

[주의]
- 개인정보 없음 (캠페인명 + 수치만)
- 처리된 파일도 폴더에 계속 유지 (재처리/디버그 용)
- 오래된 파일은 수동 삭제 권장
